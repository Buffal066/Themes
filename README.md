# JB Dark Theme

Custom dark theme with red SQL keywords.

## Installation (EASY - 2 minutes)

### Method 1: Install from Folder (Recommended)

1. **Open Cursor/VS Code**
2. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux)
3. Type: `Extensions: Install from VSIX...`
4. **OR** type: `Developer: Install Extension from Location...`
5. Navigate to this folder: `/Users/gianniperugini/repo/Themes`
6. Select the `Themes` folder
7. Reload the window when prompted

### Method 2: Manual Installation

1. **Copy the extension folder**:
   - The folder is at: `/Users/gianniperugini/repo/Themes`

2. **Open Cursor/VS Code**
3. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux)
4. Type: `Extensions: Install from VSIX...`
5. Navigate to the `Themes` folder and select it

### Method 3: Symlink (Advanced)

1. Open terminal
2. Run:
   ```bash
   ln -s /Users/gianniperugini/repo/Themes ~/.cursor/extensions/jb-dark-theme
   ```
3. Reload Cursor

## After Installation

1. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux)
2. Type: `Preferences: Color Theme`
3. Select **"JB Dark"** from the list

## Features

- Red SQL keywords (`#FF0000`)
- Custom color scheme for SQL files
- Dark theme optimized for coding
